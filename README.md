# Project Description

SmartCart UI is a responsive React-based web application that showcases a dynamic product listing and cart system. Users can browse products, view detailed features, add items to the cart, and manage their selections with real-time updates and notifications. The interface follows a Figma-based design with modern UI/UX principles.

# TechStack used
React.js
JavaScript (ES6+)
CSS / Tailwind CSS (or plain CSS if used)
React-Toastify (for notifications)
JSON (for product data handling)

# Key Features 
1. Dynamic Product & Cart Toggling
Users can switch between Product and Cart views seamlessly, with the product section shown by default.

2. Interactive Cart System
Add/remove products, view total items in the navbar, and calculate total price dynamically with real-time updates.

3. Toast Notifications & Checkout Flow
Alerts for adding/removing items and a checkout system that clears the cart using React-Toastify for better user feedback.
